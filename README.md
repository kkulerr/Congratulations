Поздравлятор (Консольное приложение)

Приложение для учета и отслеживания дней рождения с локальной базой данных.

Функционал:
 - Автоматический вывод сегодняшних и ближайших именинников при запуске.
 - Полный CRUD-цикл: добавление, редактирование и удаление записей.
 - Сортировка списка по имени, дате или количеству дней до праздника.
 - Цветовое выделение текущих и приближающихся событий в консоли.
 - Корректная обработка дат для високосных лет (29 февраля).
 - 
Стек и архитектура:
 - Технологии: C# (.NET 8), Entity Framework Core, SQLite.
 - Архитектура: Разделение на слои (Models, Data/Repositories, Services, UI).
 - Устойчивость: Данные сохраняются в локальном файле ⁠birthdays.db⁠, база создается автоматически при первом старте.
